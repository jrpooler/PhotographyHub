Create SmugMug Gallery

# Updated on 04/20/2026

  • Requires filtered Bridge collection (should be all of the photos going to SmugMug)

# Setup SmugMug for MacOS - DO THIS ONCE

  • Create an empty 'SmugMugJpegs' folder in your Documents

# Login to SmugMug website

  • Using the Control Panel Photography section, click 'SmugMug Website'
  • Enter your credentials if not already logged in
  • Click on your Username on the top row
  • On the top row, click 'Organize'
  • In the 3rd row, click '+ Create' button / Gallery
  • You should now see a 'Create Gallery'
  • Select your Preset
  • Enter Title of Gallery (like 'Fort Bragg 2026')
  • Optionally, enter Description (not required)
  • Optionally, make other adjustments (not required)
  • Double Click the 'CREATE' button
  • Minimize the SmugMug website - we will use it later

# Setup Bridge

  • Using Bridge Tools Website, double click Open Bridge (Master)
  • Click the Collection you want to send to SmugMug
  • Insure that the filters list only the photos you want to upload
  • Select all the photos in the collection (using thumbnails)
  • Navigate to Documents / LocalProjects / SmugMugDngs
  • Make certain that 'SmugMugDngs' is an empty folder
  • Right click on any of selected thumbnails / Copy
  • Open finder, navigate to Documents / LocalProjects / SmugMugDngs
  • Paste to Documents / LocalProjects / SmugMugDngs
  • Quit Bridge

# Use the Dngs to create the Jpegs

  • Using the Control Panel in the Photography section, click 'DNG to JPEG'
  • Choose the folder containing the Dngs
  • Choose the folder to place the Jpeg folder (Documents / LocalProjects / SmugMugJpegs
  • Set Maximum JPEG Size (usually around 8.0 (MB)
  • Click 'Continue' - wait for the conversion to finish!
  • Check the terminal panel for results
  • Close the terminal panel
  • Send all the Jpeg's in 'SmugMugJpegs' to the SmugMug Website
  • Navigate to Documents / LocalProjects / SmugMugJpegs
  • Select all the .jpg files in SmugMugJpegs folder
  • Open the SmugMug website that you minimized earlier (on the dock)
  • Drag all the jpegs from the desktop to the upload panel in the website
  • Wait for all jpegs to be copied
  • Click 'X' to close the uploader

# Get the Gallery Site Homepage (URL)

  • Login to your SmugMug website
  • On top row, select 'Site'
  • Scroll down to find the relevant gallery (like 'Fort Bragg 2026')
  • Double click on the picture - this will bring up the gallery
  • At the top of the gallery look for 'https://........'
  • The url should end with some random characters (for security)
  • Copy that url
  • Paste this address (url) to anyone allowed to access this gallery via email or messages

# Remove jpeg photos from SmugMugJpegs on Documents / LocalProjects / SmugMugJpegs (optional but recommended)

  • Select all jpegs
  • Right click: 'Move to Trash'
  • Leave the empty SmugMugJpegs folder in Documents / LocalProjects
